# FlowApp Landing Page

## Project Overview
This is a single-page landing page for **FlowApp**. It is a small front-end project built to practice spacing, responsive layout, and a dark visual style.

Everything lives in one `index.html` file and uses Tailwind CSS from a CDN.

## Key Features Built
* **Sticky navbar:** Simple navigation with a mobile menu.
* **Hero section:** A dark intro area with soft glow effects and a dashboard-style preview.
* **Feature grid:** A responsive card layout that works across screen sizes.
* **Testimonials:** Three quote cards with a slight offset in the middle for visual balance.
* **No extra scripts:** The page is just HTML and CSS utilities from Tailwind.

## Tech Stack
* **Markup:** Native HTML5
* **Styling:** Tailwind CSS via CDN
* **Icons:** Inline emoji and simple utility-based styling
